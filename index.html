The 3 body problem

      

The Three Body Problem is famous for its unsolvable nature. No one has or will solve the n-body problem for more than three bodies. (The general differential equations are not analytically solvable.) However, we can make a numerical approximation. This simulation allows for multiple bodies with different starting position and velocities and varying masses. It includes a tracking feature based on the distribution of mass. Try finding a stable solution (it's really difficult)!

3D projection

The image shows three squares under perspective projection. Other than calling animation frames, the entire project is written in WebAssembly. However, the API's used result in three memory operations per pixel (memory, copying to buffer, and copying to canvas). As a result, large displays will slow down and cause extreme drops in FPS. This project shows that, with reduced memory copying, 3d graphics in Wasm should be possible.
